T1048.002.02 정보유출(비대칭암호화프로토콜)

D3FEND

MITRE ATT&CK 액션을 기준으로 대응 방안을 작성

Detection

Curl과 같은 명령을 이용해 로컬에 존재하는 파일을 외부로 정송하는 행위가 발생할 경우 탐지한다.

Detection(EDR)

https://172.18.10.125:8903/#/analysis/alert/66f3daa1002dc79600004e3c

Response

  1. 이상 네트워크 트래픽 확인 및 차단 공격자가 사용하는 비대칭 암호화 프로토콜(예: SSL/TLS, SSH 등)을 통해 비정상적인 데이터 전송이 감지되었을 경우 즉시 해당 트래픽을 분석하고 차단합니다. 방화벽이나 네트워크 보안 장비를 사용하여 의심스러운 IP 주소, 도메인 또는 프로토콜에 대한 네트워크 연결을 차단합니다.

  2. 로그 분석 및 포렌식 조사 네트워크 로그 및 시스템 로그를 수집하여 공격자가 사용한 암호화 프로토콜, 유출된 데이터의 크기 및 유형, 전송된 시간대 등을 분석합니다. 암호화된 트래픽을 복호화하거나 세션 키를 확보하여 유출된 데이터의 내용을 파악할 수 있도록 시도합니다. 사용된 암호화 프로토콜의 세션 정보와 관련된 모든 로그를 확보하고, 추가적으로 유출된 정보가 있는지 분석합니다.

  3. 침해 계정 및 시스템 격리 침해된 계정 또는 시스템이 확인되면 즉시 네트워크에서 격리하고, 더 이상의 정보 유출을 방지합니다. 해당 시스템의 외부 네트워크 연결을 차단하고, 필요 시 안전한 환경에서 포렌식 조사를 진행합니다.

  4. 유출된 정보 식별 및 손해 평가 유출된 데이터의 종류를 식별하고, 민감한 정보(예: 개인정보, 금융 정보, 기밀 자료 등)가 포함되었는지 평가합니다. 유출된 데이터의 민감도에 따라 피해 규모를 평가하고, 관련 당사자 및 기관에 통보합니다.

  5. 침해 계정 비밀번호 변경 및 인증서 폐기 유출된 계정의 비밀번호를 변경하고, 관련 시스템의 접근 권한을 재설정합니다. 공격에 사용된 것으로 의심되는 SSL/TLS 인증서나 SSH 키를 폐기하고, 새로운 인증서를 발급하여 설정합니다.

  6. 외부 의심 도메인/서버 신고 및 블랙리스트 업데이트 공격자가 사용한 의심 도메인이나 IP 주소를 보안 커뮤니티나 관련 기관에 신고하고, 차단 목록(블랙리스트)에 추가합니다. 차단 목록을 보안 장비 및 보안 솔루션에 반영하여, 추가적인 공격 시도를 방지합니다.

Mitigations

  1. 암호화된 트래픽 모니터링

    • SSL/TLS, SSH와 같은 암호화된 트래픽에 대해 딥 패킷 인스펙션(DPI)을 사용하여 비정상적인 데이터 유출을 감지합니다.
    • 네트워크 보안 장비를 사용하여 암호화 트래픽을 실시간으로 모니터링하고, 의심스러운 패턴이 발견되면 경고를 생성하도록 설정합니다.

  2. 암호화 프로토콜 정책 강화

    • 내부 네트워크에서 사용되는 암호화 프로토콜에 대해 정책을 강화하고, 허용된 암호화 프로토콜만 사용하도록 제한합니다.
    • 약한 암호화 알고리즘이나 프로토콜(예: SSL, 취약한 SSH 설정 등)을 차단하고, 최신 보안 기준에 맞는 강력한 암호화 프로토콜만 허용합니다.

  3. 유출 방지 솔루션(DLP) 도입

    • 데이터 유출 방지(DLP) 솔루션을 도입하여, 민감한 정보가 암호화 프로토콜을 통해 외부로 전송되는 경우 이를 탐지하고 차단합니다.
    • DLP 솔루션을 통해 중요한 데이터의 이동을 실시간으로 모니터링하고, 규칙을 설정하여 특정 데이터를 암호화된 트래픽으로 전송하는 것을 방지합니다.

  4. 로그 및 알림 설정 강화

    • 암호화 프로토콜을 사용한 데이터 전송 시 관련 로그를 상세하게 기록하고, 비정상적인 트래픽 발생 시 보안 담당자에게 즉시 알림이 가도록 설정합니다.
    • 로그 수집 및 분석을 통해 의심스러운 활동을 빠르게 탐지하고 대응할 수 있도록 자동화된 경고 시스템을 구축합니다.

  5. 보안 교육 및 인식 제고

    • 암호화 프로토콜을 이용한 정보 유출에 대한 보안 교육을 실시하고, 직원들이 이러한 위협에 대해 인식할 수 있도록 합니다.
    • 공격자가 주로 사용하는 암호화 프로토콜을 통해 정보를 유출하는 방법과 대응 방안에 대해 주기적으로 교육하고, 보안 인식을 제고합니다.
