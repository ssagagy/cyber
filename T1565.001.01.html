T1565.001.01 Data Manipulation- Stored Data Maniqulation

D3FEND

MITRE ATT&CK 액션을 기준으로 대응 방안을 작성

Detection

  • CurrentProcess = "powershell.exe" AND
  • Action = "Amsi_Scan" AND
  • AmsiScandata = "rd" AND "-recurse"

    Detection(EDR)

Response

공격자에 의해 PC가 제어되었을 가능성이 있으므로 호스트 네트워크를 격리합니다.

Mitigations

  1. 데이터 무결성 검증 및 모니터링

    • 파일 무결성 모니터링(File Integrity Monitoring, FIM) 도구를 사용하여 중요 데이터의 변조 여부를 감지합니다.
    • 주기적으로 해시 값(SHA-256, MD5 등)을 생성하여 데이터 변경 여부를 확인합니다.

  2. 강력한 접근 제어 및 권한 관리

    • 중요 데이터에 대한 최소 권한 원칙(Least Privilege) 을 적용하여 불필요한 쓰기 권한을 제한합니다.
    • 다중 인증(MFA)을 적용하여 무단 접근을 방지합니다.

  3. 정기적인 백업 및 복구 계획 수립

    • 중요 데이터를 주기적으로 백업하고, 백업 파일이 변조되지 않았는지 검증합니다.
    • 오프라인 또는 변경 불가능한(Immutable) 백업을 유지하여 공격으로부터 보호합니다.

  4. 로그 및 감사 기능 강화

    • 데이터 변경 이벤트를 로깅하고, 이상 징후를 분석하여 탐지합니다.
    • SIEM(Security Information and Event Management) 솔루션을 활용하여 실시간 분석을 수행합니다.

  5. 애플리케이션 및 데이터베이스 보안 강화

    • 데이터베이스 암호화를 사용하여 무단 변경을 방지합니다.
    • 데이터 변경 요청을 처리하는 애플리케이션에서 입력 검증(Input Validation)을 강화하여 조작된 데이터를 차단합니다.
