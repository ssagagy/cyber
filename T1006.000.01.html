T1006.000.01 볼륨 직접 접근

D3FEND

MITRE ATT&CK 액션을 기준으로 대응 방안을 작성

Detection

None

Detection(EDR)

None

Response

디스크에 비정상적으로 접근한 프로세스나 계정이 발견되면 즉시 격리하여 추가적인 공격을 방지합니다.

Mitigations

  1. 패스워드 정책 강화

    • 강력한 패스워드 정책을 적용하여 패스워드 길이, 복잡성, 유효 기간을 설정합니다. 예를 들어, 최소 12자리 이상의 복잡한 패스워드를 요구하고 대소문자, 숫자, 특수문자를 포함하도록 설정합니다.
    • 그룹 정책(GPO)를 통해 패스워드 정책을 설정하고 Password Policy에서 복잡성 요구 사항을 활성화합니다.

  2. 보안 솔루션 강화

    • EDR/XDR 솔루션을 활용하여 패스워드 필터에 의심스러운 변경이나 악성 코드 설치를 탐지하고 차단합니다.
    • Windows Defender 및 타 보안 소프트웨어에서 패스워드 필터 변경을 감지하는 기능을 활성화합니다.

  3. 이벤트 로깅 및 감사 정책 강화

    • Windows Event Log에서 패스워드 변경 및 로그인 시도에 대한 로그(4723, 4724, 4725, 4726)를 모니터링합니다.
    • 패스워드 필터 설정이 변경되었을 때 즉시 경고를 발생시킬 수 있도록 SIEM 시스템을 설정합니다.

  4. 권한 제어

    • 패스워드 필터를 변경할 수 있는 관리자 권한을 최소화합니다. 패스워드 필터를 수정하려면 반드시 인증된 관리자만 수행할 수 있도록 권한을 제한합니다.
    • Windows 권한 강화 및 최소 권한 원칙을 적용하여 시스템에 대한 액세스를 제한합니다.

  5. 레지스트리 및 시스템 설정 보호

    • 패스워드 필터가 변경되지 않도록 레지스트리 보호를 활성화합니다.
    • 그룹 정책을 통해 패스워드 필터의 등록을 제한하거나 변경할 수 없도록 설정합니다.

  6. 정기적인 보안 점검 및 패치 적용

    • 정기적으로 시스템을 점검하고 패스워드 필터 및 보안 설정이 의도한 대로 유지되고 있는지 확인합니다.
    • 최신 보안 패치를 적용하여 패스워드 필터 우회를 위한 알려진 취약점을 차단합니다.
