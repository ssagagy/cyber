T1588.002.01 PowerShell을 사용하여 오픈소스웹쉘(RedHacker)다운로드

D3FEND

MITRE ATT&CK 액션을 기준으로 대응 방안을 작성

Detection

알려진 C2 또는 다운로드 받은 악성코드의 페이로드로 탐지합니다.

Detection(EDR)

행동 유형 : 네트워크 접속

오퍼레이션 : contains

값 : <알려진 C2 또는 다운로드 받는 악성코드의 페이로드>

https://172.18.10.125:8903/#/analysis/incident/67a9a5ce002dc6ce00000eae

Response

  • 의심스러운 프로세스 격리 및 실행 차단
  • 침해된 시스템을 분석하여 추가적인 악성코드 감염 여부 확인
  • 공격자가 활용한 네트워크 및 계정 접근 권한 제거
  • 패치 및 보안 업데이트 적용하여 재발 방지

Mitigations

  1. EDR/XDR 솔루션을 통한 탐지 및 차단

  • 엔드포인트 보안 솔루션(EDR/XDR) 적용하여 악성코드 실행 탐지 및 차단

  • Cobalt Strike, Mimikatz, Metasploit 등의 해킹 도구 실행 탐지 및 차단 규칙 설정

  • 파일 무결성 모니터링(FIM) 적용하여 의심스러운 실행 파일 탐지

  1. 네트워크 및 웹 필터링 적용

  • SIEM(Security Information and Event Management) 솔루션을 사용하여 다크웹, 해킹 포럼 접속 탐지

  • 보안 프록시 또는 DNS 필터링을 활용하여 악성코드 배포 사이트 차단

  • Firewall 및 IDS/IPS에서 악성코드 다운로드 시도 차단 (YARA, Snort 규칙 적용)

  1. 이메일 및 파일 보안 강화

  • 첨부 파일 스캔 및 실행 파일(.exe, .dll, .scr) 자동 차단

  • 이메일 피싱 방어를 위해 DMARC, DKIM, SPF 설정

  • USB 및 외부 저장 장치 사용 제한 (DLP 솔루션 적용)

  1. 소프트웨어 무결성 검증 및 제한

  • AppLocker, WDAC(Windows Defender Application Control)로 승인된 소프트웨어만 실행 가능하도록 설정

  • PowerShell 실행 정책 제한 (Set-ExecutionPolicy Restricted)

  • Windows Script Host (WSH) 및 매크로 실행 차단
