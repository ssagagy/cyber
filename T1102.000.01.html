### Detection 난독화된 스크립트의 파일 다운로드 명령어(DownloadFile)를 탐지 했습니다.

### Detection(EDR)

### Response 공격자 C&C 서버로 추정되는 주소를 차단합니다.

### Mitigations 1. 네트워크 트래픽 분석 (M1049 - Audit) - HTTP 및 HTTPS 트래픽을 모니터링하고, 비정상적인 웹 서비스 통신을 탐지할 수 있도록 네트워크 로그를 분석합니다.

  1. 필터링 및 방화벽 설정 강화 (M1030 - Network Segmentation)

    • 웹 서비스에 대한 트래픽을 필터링하고, 방화벽 규칙을 설정하여 신뢰할 수 있는 도메인 및 서비스로의 접근만 허용합니다.

  2. 도메인 허용 목록 관리 (M1037 - Filter Network Traffic)

    • 신뢰할 수 있는 웹 서비스의 도메인만 허용하도록 네트워크 트래픽을 제어하고, 악성 웹 서비스와의 통신을 차단합니다.

  3. 명령 및 제어 트래픽 탐지 (M1017 - Network Intrusion Prevention)

    • 웹 서비스 기반 C2 통신을 탐지하기 위해 네트워크 침입 방지 시스템(IPS) 및 침입 탐지 시스템(IDS)을 설정하여, 의심스러운 HTTP/HTTPS 트래픽을 탐지하고 차단합니다.

  4. 웹 서비스 사용 정책 제한 (M1021 - Restrict Web-Based Content)

    • 조직 내에서 사용할 수 있는 웹 서비스와 도구를 제한하여, 공격자가 웹 서비스 기반의 명령 및 제어 통신을 설정하는 것을 방지합니다.
