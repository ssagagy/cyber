T1486.000.01 Data Encrypted for Impact

D3FEND

MITRE ATT&CK 액션을 기준으로 대응 방안을 작성

Detection

탐지 방법 작성, EFR과 무관하게 해당 Action에 대한 탐지 방법 명시

Detection(EDR)

Response

  • 공격자에 의해 PC가 제어되었을 가능성이 있으므로 호스트 네트워크를 격리합니다.
  • 공격자 C&C 서버로 추정되는 주소를 차단합니다.
  • 해당 행위를 수행한 프로세스를 종료합니다

Mitigations

1. 정기적인 데이터 백업 및 복구 전략 수립

  • 3-2-1 백업 전략을 적용하여 오프라인 및 변경 불가능한(Immutable) 백업을 유지합니다.
  • 백업 데이터를 별도의 네트워크에 격리하여 랜섬웨어 감염을 방지합니다.
  • 정기적인 백업 복구 테스트를 수행하여 실제 공격 발생 시 신속한 복구가 가능하도록 합니다.

2. 최소 권한 원칙 적용 및 네트워크 격리

  • 사용자 및 서비스 계정에 최소 권한 원칙(Least Privilege) 을 적용하여 불필요한 파일 접근을 차단합니다.
  • 중요한 데이터 저장소 및 시스템을 네트워크에서 격리하여 랜섬웨어 확산을 방지합니다.
  • 네트워크 세분화(Network Segmentation) 를 통해 감염된 시스템이 전체 네트워크에 영향을 미치지 않도록 합니다.

3. 파일 무결성 모니터링 및 이상 탐지

  • File Integrity Monitoring(FIM) 을 사용하여 파일 암호화 시도를 실시간으로 탐지합니다.
  • SIEM(Security Information and Event Management) 솔루션을 활용하여 대량 파일 변경 또는 암호화 시도를 감지하고 경고를 생성합니다.
  • Windows 이벤트 로그 및 Sysmon 을 설정하여 비정상적인 파일 변경 및 프로세스 활동을 감시합니다.

4. 랜섬웨어 방어 기능 적용

  • Windows Defender Controlled Folder Access 기능을 활성화하여 중요 폴더를 보호합니다.
  • EDR(Endpoint Detection and Response) 및 NGAV(Next-Generation Antivirus) 솔루션을 사용하여 랜섬웨어 행위를 탐지하고 차단합니다.
  • 랜섬웨어가 C2(Command and Control) 서버와 통신하지 못하도록 DNS 필터링 및 방화벽 정책을 강화합니다.

5. 사용자 교육 및 피싱 방어

  • 피싱 이메일 시뮬레이션 훈련을 통해 사용자가 악성 첨부파일이나 링크를 열지 않도록 교육합니다.
  • 정기적인 보안 인식 교육을 통해 의심스러운 이메일, 다운로드 및 실행 파일을 신중하게 취급하도록 유도합니다.
